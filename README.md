📌 Project Overview

This system has two main parts:

1. WhatsApp Bot (Patient Side)
- Patient messages hospital on WhatsApp
- Selects department and doctor
- Books appointment with date and time
- Pays via Cash or Bank Transfer
- Receives instant confirmation
- Can view and cancel appointments
- No app download needed

2. Admin Dashboard (Hospital Side)
- View all appointments
- Confirm or cancel bookings
- Manage doctors and departments
- Track payments and revenue
- View patient records
- Update hospital settings dynamically
